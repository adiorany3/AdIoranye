# Patch Tampilan Kode Web

Patch ini memperbaiki tampilan kode di web Streamlit:

- Blok kode tampil seperti editor gelap.
- Kode panjang tidak keluar dari bubble chat.
- Kode memakai horizontal scroll agar indentasi tetap rapi.
- Inline code dibuat lebih jelas.
- Tabel/dataframe diberi overflow-x agar tidak merusak layout.
- Mobile lebih nyaman dibaca.

File utama yang berubah: app.py
